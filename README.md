# Hope Kuku Shop 🍗

A professional e-commerce platform for Hope Kuku Shop, specializing in premium roasted chicken.

## Features
- **Modern UI/UX**: Built with a clean, responsive design.
- **Full Menu**: Categorized menu with filtering (Chicken, Sides).
- **Cart System**: LocalStorage-based cart for a seamless experience.
- **Robust Checkout**: Captures delivery details and supports M-Pesa and Bitcoin.
- **Admin Dashboard**: View and manage customer orders and statuses.
- **Backend**: Flask-based API with SQLAlchemy and SQLite database.

## Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python app.py
   ```
4. Access the shop at `http://localhost:5000`.

## Admin Access
View orders at `http://localhost:5000/admin/orders`.
